# KOLAMBA RUN

A Sri Lankan meme-inspired 2D endless runner built with **HTML5 Canvas, CSS3, and JavaScript**.

> **Run. Dodge. Collect. Survive.**

---

## About

**KOLAMBA RUN** is a browser-based arcade survival game inspired by Sri Lankan road chaos and local meme culture.

Run through busy roads, dodge vehicles and enemy attacks, collect Sri Lankan currency notes, use power-ups, build combos, and survive for as long as possible.

---

## Features

- Sri Lankan-themed road environment
- Endless runner gameplay
- Desktop and mobile support
- Mobile landscape mode
- Easy, Medium, and Hard difficulties
- Jump, double-jump, and slide mechanics
- Enemy shooting system
- Health and combo systems
- Near-miss bonuses
- Dynamic road events
- Sri Lankan vehicle obstacles
- Currency collectibles
- Multiple power-ups
- Background music and sound effects
- Pause and mute controls
- Responsive arcade-style UI

---

## Obstacles

- Sri Lankan Bus
- CT100 Bike
- Tuk-Tuk
- Lorry
- Potholes
- Low road signs
- Enemy bullets

---

## Power-Ups

| Power-Up | Effect |
|---|---|
| Shield | Protects the player from damage |
| Magnet | Attracts nearby currency notes |
| Slow Time | Temporarily slows the game |
| Extra Heart | Restores health |
| 2x Score | Temporarily doubles score gain |

---

## Controls

### Desktop

| Action | Control |
|---|---|
| Jump | `Space` or `Arrow Up` |
| Slide | `Arrow Down` |
| Pause | `P` |

### Mobile

Play in **landscape orientation**.

```text
LEFT   →   JUMP
RIGHT  →   SLIDE
```

## Run Locally

Clone the repository:

```bash
git clone https://github.com/D3NUW4NXD/kolamba-run.git
```

Enter the project folder:

```bash
cd kolamba-run
```

Open it in Visual Studio Code:

```bash
code .
```

Then run `index.html` using Live Server or open it directly in your browser.
